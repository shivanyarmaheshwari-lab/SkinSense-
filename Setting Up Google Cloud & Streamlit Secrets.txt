3. Setting Up Google Cloud & Streamlit Secrets
To make the code above work, you need to configure the connection between Google and Streamlit.

Step A: Get Google Credentials

Go to the Google Cloud Console.

Create a new Project (e.g., "Leukemia-App").

Enable two APIs for this project: Google Drive API and Google Sheets API.

Go to IAM & Admin > Service Accounts. Create a new Service Account.

Generate a JSON Key for this Service Account and download it. It contains your client_email and private_key.

Step B: Prepare Google Drive & Sheets

Google Drive: Create a folder where the images should go. Right-click the folder, select Share, and paste the client_email from your JSON key. Give it "Editor" access.

Find the Folder ID: Look at the URL of the folder: drive.google.com/drive/folders/YOUR_FOLDER_ID.

Google Sheets: Create a new blank Google Sheet. Share it with the same client_email and give it "Editor" access. Add headers to the first row (Date, Patient ID, Age, Gender, Prediction, Confidence, Agreement, Notes, Drive ID).

Find the Spreadsheet ID: Look at the URL: docs.google.com/spreadsheets/d/YOUR_SPREADSHEET_ID/edit.

Step C: Configure Streamlit Secrets
Do not upload your JSON key to GitHub. Instead, when you deploy the app on share.streamlit.io, go to your App Settings -> Secrets, and format your JSON key and IDs like this:

[gcp_service_account]
type = "service_account"
project_id = "your-project-id"
private_key_id = "your-key-id"
private_key = "-----BEGIN PRIVATE KEY-----\nYOUR_LONG_KEY_HERE\n-----END PRIVATE KEY-----\n"
client_email = "your-service-account@your-project.iam.gserviceaccount.com"
client_id = "your-client-id"
auth_uri = "https://accounts.google.com/o/oauth2/auth"
token_uri = "https://oauth2.googleapis.com/token"
auth_provider_x509_cert_url = "https://www.googleapis.com/oauth2/v1/certs"
client_x509_cert_url = "your-cert-url"

# Add your IDs here so the app can find them
spreadsheet_id = "YOUR_SPREADSHEET_ID"
drive_folder_id = "YOUR_FOLDER_ID"
